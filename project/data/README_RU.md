# Директория Данных

Эта директория содержит датасеты и документы для системы Research Assistant.

## Текущие Датасеты

### Anthropic HH-RLHF Dataset
**Источник**: https://huggingface.co/datasets/Anthropic/hh-rlhf

**Описание**:
- Данные о человеческих предпочтениях касательно полезности и безопасности
- Содержит ~170K диалогов с аннотациями людей
- Полезен для обучения и оценки разговорных AI систем
- Red-teaming данные для исследований безопасности

**Использование в Неделе 7**:
Этот датасет будет использоваться для демонстрации RAG (Retrieval-Augmented Generation) системы:
1. Загрузка диалогов из датасета
2. Создание эмбеддингов и сохранение в векторной БД (pgvector)
3. Реализация поиска по сходству для релевантных диалогов
4. Использование извлеченного контекста для улучшения ответов

## Инструкции по Установке

### Вариант 1: Загрузка через Hugging Face Datasets

```bash
pip install datasets
```

```python
from datasets import load_dataset

# Загрузить датасет
dataset = load_dataset("Anthropic/hh-rlhf")

# Сохранить локально

```

### Вариант 2: Использовать готовый скрипт

```bash

python scripts/download_dataset.py

python scripts/download_dataset.py --subset 1000

# Показать статистику датасета
python scripts/download_dataset.py --stats
```

### Пример Диалога из Датасета

Датасет содержит диалоги в формате:

```json
{
  "chosen": "Human: Что такое машинное обучение?\n\nAssistant: Машинное обучение - это...",
  "rejected": "Human: Что такое машинное обучение?\n\nAssistant: Я не знаю..."
}
```

- `chosen` - предпочтительный ответ (полезный и безопасный)
- `rejected` - менее предпочтительный ответ

## Размер Датасета

- **Полный датасет**: ~500MB-1GB
- **Примеры**: ~170,000 диалогов
- **Splits**: train, test

Для тестирования рекомендуется использовать подмножество (1000-5000 примеров).

## Заметки

- Датасет содержит английские диалоги
- Некоторые диалоги содержат обсуждение чувствительных тем (red-teaming)
